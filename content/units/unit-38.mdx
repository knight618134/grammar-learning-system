## Learning Goal
Choose between real conditionals and unreal conditionals.

## Core Grammar
Use **if + present, will/can/may + base verb** for real future possibilities. Use **if + past, would/could/might + base verb** for unreal or less likely situations.

## Examples
- If I review tonight, I will remember the passive patterns better.
- If I had more time, I would do another test.
- If this sentence is passive, the verb needs **be + past participle**.

## Jason's Confusing Points
- The past form after **if** can describe an unreal present or future, not only the past.
- Do not use **would** in the if-clause for the basic second conditional.

## Wrong Answers
- Wrong: If I would have time, I would review.
- Correct: If I had time, I would review.
- Wrong: If I will study, I will improve.
- Better: If I study, I will improve.

## One-line Summary
Use **if + present** for real possibilities and **if + past** for unreal situations.
