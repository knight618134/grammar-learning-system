## Learning Goal
EN: Use **would** for imagined situations, polite language, preferences, and repeated past actions.

中：學會用 **would** 表達假設、禮貌說法、偏好，以及過去重複動作。

## Core Grammar
### A. Would In Unreal Situations / 假設中的 would
EN: Use **would + base verb** for imagined results.

中：用 **would + 原形動詞** 表達假設情境中的結果。

- If I had more time, I would review more.

### B. Would Like / 禮貌想要
EN: **Would like** is a polite way to say want.

中：**would like** 是比較禮貌的「想要」。

- I would like more examples.

### C. Would For Past Habits / 過去重複動作
EN: **Would** can describe repeated actions in the past, but not usually past states.

中：**would** 可以描述過去常做的動作，但通常不用來描述狀態。

- When I was younger, I would memorize grammar rules.

### D. Polite Requests / 禮貌請求
EN: **Would you...?** can make requests more polite.

中：**Would you...?** 可以讓請求更禮貌。

- Would you explain this again?

## Examples
- I would review more if I had time.
- Would you explain this sentence again?
- I would like a Chinese explanation.
- When I was younger, I would memorize grammar rules.

## Jason's Confusing Points
- EN: **Would** is not always future-in-the-past.
- 中：**would** 不只是一種「過去的未來」。
- EN: Use **used to** more naturally for past states.
- 中：描述過去狀態時，**used to** 通常比 would 自然。

## Wrong Answers
- Wrong: I would to practice more.
- Correct: I would practice more.
- Wrong: I would be shy when I was a child.
- Better: I used to be shy when I was a child.

## One-line Summary
EN: **Would** is useful for imagined results, polite language, and repeated past actions.

中：**would** 常用於假設結果、禮貌說法、過去重複動作。
