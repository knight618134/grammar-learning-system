## Learning Goal
EN: Use present perfect continuous for actions continuing until now or recently.

中：用現在完成進行式表達一直持續到現在或最近剛在做的事。

## Core Grammar
### A. Form / 形式
EN: have/has been + -ing

中：have/has been + V-ing

### B. Continuing Action / 持續動作
EN: Use it for actions that started before now and continue.

中：從過去開始，持續到現在。

### C. Recent Activity / 最近活動
EN: Use it when recent activity explains the present result.

中：最近做的事造成現在結果。

## Examples
- I have been reviewing grammar.
- Jason has been building this app.
- It has been getting easier.

## Jason's Confusing Points
- This form emphasizes activity or duration.
- 中：這個時態強調活動本身或持續多久。

## Wrong Answers
- Wrong: I have been review grammar.
- Correct: I have been reviewing grammar.

## One-line Summary
EN: Present perfect continuous = have/has been + -ing.

中：本單元重點請看上方公式、例句和常錯點；複習時先確認時態和動詞形式。
