## Learning Goal
Use **would** for imagined situations, polite requests, preferences, and repeated past actions.

## Core Grammar
**Would + base verb** can show hypothetical meaning. **Would like** is polite. **Would often** can describe repeated past actions.

## Examples
- I would review more if I had time.
- Would you explain this sentence again?
- When I was younger, I would memorize grammar rules.

## Jason's Confusing Points
- **Would** is not always future-in-the-past.
- For past habits, **used to** can describe states, but **would** usually describes repeated actions.

## Wrong Answers
- Wrong: I would to practice more.
- Correct: I would practice more.
- Wrong: I would be shy when I was a child.
- Better: I used to be shy when I was a child.

## One-line Summary
Use **would** for imagined actions, polite language, and repeated past actions.
