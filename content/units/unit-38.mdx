## Learning Goal
EN: Choose between real conditionals and unreal conditionals.

中：學會分辨「真實可能發生」和「假設、不太可能、與現在不同」的 if 句。

## Core Grammar
### A. Real Condition: if + present / 真實條件
EN: Use **if + present** when the situation is real, possible, or likely.

中：如果事情是真實可能發生的，用 **if + 現在式**。

- If I review tonight, I will remember it better.
- If this sentence is passive, it needs **be + past participle**.
- If you practice every day, you can improve.

### B. Unreal Condition: if + past / 假設條件
EN: Use **if + past** when the situation is unreal, imagined, or less likely.

中：如果事情是假設的、和現在不同的、或不太可能發生，用 **if + 過去式**。

- If I had more time, I would review again.
- If I knew the answer, I would explain it.
- If I were better at grammar, I would make fewer mistakes.

### C. Do Not Put Would In The If-Clause / if 子句不要放 would
EN: In the basic second conditional, use **if + past**, not **if + would**.

中：基本第二條件句裡，if 後面用過去式，不要寫 would。

- Wrong: If I would have time, I would review.
- Correct: If I had time, I would review.

### D. Meaning, Not Time / 看意思，不只看時間
EN: The past form after **if** can describe an unreal present or future. It does not always mean past time.

中：if 後面的過去式常常表示「假設」，不一定是真的過去時間。

- If I knew the answer now, I would tell you.
- If I had time tomorrow, I would review Unit 43.

## Examples
- If I review tonight, I will remember the passive patterns better.
- If I had more time, I would do another test.
- If this sentence is passive, the verb needs **be + past participle**.
- If I understood this rule, I would not choose the wrong form.

## Jason's Confusing Points
- EN: **If I do** = real possibility.
- 中：**If I do** 是真的有可能。
- EN: **If I did** = unreal or less likely.
- 中：**If I did** 是假設、和現在不同、或比較不可能。
- EN: Do not translate word by word from Chinese; check whether the condition is real or imagined.
- 中：不要從中文逐字翻，先判斷條件是真的可能，還是假設。

## Wrong Answers
- Wrong: If I would have time, I would review.
- Correct: If I had time, I would review.
- Wrong: If I will study, I will improve.
- Better: If I study, I will improve.
- Wrong: If I am you, I would review Passive 2.
- Correct: If I were you, I would review Passive 2.

## One-line Summary
EN: Use **if + present** for real possibilities and **if + past** for unreal situations.

中：真實可能用 **if + 現在式**；假設情況用 **if + 過去式**。
