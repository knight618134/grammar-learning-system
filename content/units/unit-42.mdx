## Learning Goal
Use advanced passive patterns, including continuous passive, perfect passive, and get passive.

## Core Grammar
Use **be being + past participle** for an action in progress. Use **have/has/had been + past participle** for perfect passive. **Get + past participle** is common in informal speech.

## Examples
- The room is being cleaned now.
- The report has been checked twice.
- My phone got damaged yesterday.

## Jason's Confusing Points
- **Was being cleaned** means the cleaning was in progress.
- **Had been cleaned** means the cleaning was already complete before another past time.

## Wrong Answers
- Wrong: The room was being cleaned before we arrived.
- Better: The room had been cleaned before we arrived.
- Wrong: The file has checked.
- Correct: The file has been checked.

## One-line Summary
Use continuous passive for ongoing action and perfect passive for completed action before a reference point.
