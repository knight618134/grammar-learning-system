## Learning Goal
EN: Use passive forms after modal verbs, perfect forms, and continuous forms.

中：學會進階被動：**be done**、**been done**、**being done**，尤其是 modal、完成式、進行式裡的被動。

## Core Grammar
### A. Modal Passive: be done / 情態助動詞後面的被動
EN: After modal verbs, passive uses **modal + be + past participle**.

中：在 should / can / must / may / might 這些情態助動詞後面，被動要用 **be + 過去分詞**。

- The sentence should **be corrected**.
- This rule must **be reviewed**.
- The answer can **be changed**.

### B. Perfect Passive: been done / 完成式被動
EN: In perfect passive, use **have/has/had been + past participle**.

中：完成式被動要用 **have/has/had been + 過去分詞**。看到 **been**，通常代表它在完成式裡。

- The report has **been checked**.
- The mistake had **been corrected** before I noticed it.

### C. Continuous Passive: being done / 進行式被動
EN: In continuous passive, use **be being + past participle**.

中：進行式被動要用 **be being + 過去分詞**。意思是「正在被...」。

- The room is **being cleaned** now.
- The test was **being marked** when I arrived.

### D. How to Choose / 怎麼選
EN: Ask what meaning you need: modal possibility/necessity, completed action, or action in progress.

中：先問自己要表達哪一種意思：可能/必要、已經完成、還是正在進行。

- should be corrected = 應該被改
- has been corrected = 已經被改
- is being corrected = 正在被改

## Examples
- The sentence should be corrected.
- The report has been checked twice.
- The room is being cleaned now.
- The answer had been marked before I noticed the mistake.
- This exercise can be reviewed again tomorrow.
- My answer has been saved in the wrong-answer list.

## Jason's Confusing Points
- **Be done** appears after modals or infinitive-like structures.
- **Been done** appears in perfect passive forms.
- **Being done** means the passive action is in progress.
- 中：**be / been / being** 不是隨便選，它們各自代表不同文法位置。
- 中：如果前面有 should / can / must，後面通常是 **be done**，不是直接接 done。
- 中：如果有 has / have / had，就要想到 **been done**。

## Wrong Answers
- Wrong: The file has checked.
- Correct: The file has been checked.
- Wrong: The room is cleaned now when the action is happening now.
- Better: The room is being cleaned now.
- Wrong: This should corrected.
- Correct: This should be corrected.
- Wrong: The mistake has corrected.
- Correct: The mistake has been corrected.
- Wrong: The sentence is being correct.
- Correct: The sentence is being corrected.

## One-line Summary
EN: Use **be done**, **been done**, and **being done** to show passive meaning in different verb patterns.

中：**be done**、**been done**、**being done** 的差別，就是它們出現在不同文法結構裡。
