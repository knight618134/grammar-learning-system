## Learning Goal
Use modal verbs naturally for requests, offers, permission, and invitations.

## Core Grammar
**Can**, **could**, and **would** can make interaction more polite. **Could you...?** and **Would you...?** are common request patterns. **Can I...?** and **Could I...?** ask for permission.

## Examples
- Could you check this sentence?
- Would you explain the difference again?
- Can I use the passive here?
- Would you like to review Unit 43 first?

## Jason's Confusing Points
- **Could you...?** asks someone else to do something.
- **Could I...?** asks permission for yourself.
- **Would you like...?** is an invitation or offer, not a request for help.

## Wrong Answers
- Wrong: Could I check this for me?
- Better: Could you check this for me?
- Wrong: Would you like explain this?
- Correct: Would you like me to explain this?

## One-line Summary
Use the modal pattern to match the social function: request, offer, permission, or invitation.
