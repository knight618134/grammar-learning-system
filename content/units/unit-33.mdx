## Learning Goal
EN: Use **should** and **shouldn't** for advice, opinions, expected situations, and sensible choices.

中：學會用 **should / shouldn't** 表達建議、看法、預期，以及「比較合理的做法」。

## Core Grammar
### A. Should For Advice / should 表達建議
EN: Use **should + base verb** to say what is a good or sensible thing to do.

中：用 **should + 原形動詞** 表達「應該做、這樣比較好」。

- You should review the examples.
- You shouldn't rush through the test.

### B. Should For Opinion / should 表達看法
EN: **Should** can show what the speaker thinks is right or reasonable.

中：**should** 也可以表達說話者覺得合理、正確的判斷。

- This answer should be correct.
- The passive form should include **be**.

### C. Should Is Weaker Than Must / should 比 must 弱
EN: **Should** is advice. **Must** is stronger obligation.

中：**should** 是建議；**must** 是比較強的必須。

- You should review this. = 建議你複習
- You must review this. = 你必須複習

### D. Form Check / 形式檢查
EN: After **should**, use the base verb. Do not use **to**.

中：**should** 後面接原形動詞，不加 **to**。

- should review
- should be corrected
- should not ignore

## Examples
- You should review the confusing points before doing new exercises.
- This answer should be correct, but check the tense again.
- You shouldn't translate every sentence word by word.
- Jason should save useful ChatGPT explanations into the notes.

## Jason's Confusing Points
- EN: **Should** is not as strong as **must**.
- 中：**should** 沒有 **must** 那麼強。
- EN: **Should be done** is modal passive: should + be + past participle.
- 中：**should be done** 是情態助動詞被動：should + be + 過去分詞。

## Wrong Answers
- Wrong: You should to practice more.
- Correct: You should practice more.
- Wrong: This should to be correct.
- Correct: This should be correct.
- Wrong: This should corrected.
- Correct: This should be corrected.

## One-line Summary
EN: **Should + base verb** gives advice or shows what is expected.

中：**should + 原形動詞** 表達建議或預期。
