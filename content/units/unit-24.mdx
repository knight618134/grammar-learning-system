## Learning Goal
EN: Use future continuous and future perfect.

中：學會使用未來進行式和未來完成式。

## Core Grammar
### A. Will Be Doing / 未來某時正在
EN: Use will be doing for an action in progress at a future time.

中：未來某時間正在做，用 will be doing。

### B. Will Have Done / 未來某時已完成
EN: Use will have done for completion before a future time.

中：未來某時間之前完成，用 will have done。

### C. Time Point / 時間點
EN: These forms often need a future reference point.

中：這些時態常需要未來時間點。

## Examples
- At 9, I will be reviewing grammar.
- By tomorrow, I will have finished Unit 24.
- Jason will have built a strong review system.

## Jason's Confusing Points
- Be doing = in progress; have done = completed.
- 中：be doing 是正在；have done 是完成。

## Wrong Answers
- Wrong: By tomorrow, I will finish already.
- Better: By tomorrow, I will have finished.

## One-line Summary
EN: Will be doing = future in progress; will have done = completed before future time.

中：本單元重點請看上方公式、例句和常錯點；複習時先確認時態和動詞形式。
