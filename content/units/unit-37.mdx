## Learning Goal
EN: Use modal verbs naturally for requests, offers, permission, and invitations.

中：學會用 can / could / would 做請求、邀請、允許、提供幫助。

## Core Grammar
### A. Requests / 請別人幫忙
EN: Use **Can/Could/Would you...?** to ask someone to do something.

中：請別人幫忙時，用 **Can/Could/Would you...?**。

- Could you check this sentence?
- Would you explain the difference again?

### B. Permission / 問可不可以
EN: Use **Can/Could I...?** to ask for permission.

中：問自己可不可以做某事，用 **Can/Could I...?**。

- Can I use the passive here?
- Could I ask one more question?

### C. Offers / 提供幫忙
EN: Use **Can I...?** or **Shall I...?** to offer help.

中：主動提供幫忙時，可以用 **Can I...? / Shall I...?**。

- Can I help you review this?

### D. Invitations / 邀請
EN: Use **Would you like...?** for invitations or offers.

中：邀請或提供東西時，用 **Would you like...?**。

- Would you like to review Unit 43 first?

## Examples
- Could you check this sentence?
- Would you explain the difference again?
- Can I use the passive here?
- Would you like to review Unit 43 first?

## Jason's Confusing Points
- EN: **Could you...?** asks another person to do something.
- 中：**Could you...?** 是請對方做事。
- EN: **Could I...?** asks permission for yourself.
- 中：**Could I...?** 是問自己可不可以。
- EN: **Would you like...?** is an invitation or offer.
- 中：**Would you like...?** 是邀請或提供，不是請求對方幫忙。

## Wrong Answers
- Wrong: Could I check this for me?
- Better: Could you check this for me?
- Wrong: Would you like explain this?
- Correct: Would you like me to explain this?
- Wrong: Can you to help me?
- Correct: Can you help me?

## One-line Summary
EN: Match the modal pattern to the social function: request, permission, offer, or invitation.

中：先判斷功能：請求、允許、提供、邀請，再選 modal 句型。
