## Learning Goal
EN: Use **should have done** and **shouldn't have done** to talk about better choices in the past.

中：學會用 **should have done / shouldn't have done** 表達「過去本來應該做/不該做」。

## Core Grammar
### A. Should Have Done / 本來應該做
EN: **Should have + past participle** means the better action was not done.

中：**should have + 過去分詞** 表示過去沒做，但本來應該做。

- I should have checked the tense.
- I should have saved that explanation.

### B. Shouldn't Have Done / 本來不該做
EN: **Shouldn't have + past participle** means the action happened, but it was not a good idea.

中：**shouldn't have + 過去分詞** 表示做了，但本來不該做。

- I shouldn't have rushed.
- I shouldn't have ignored the passive form.

### C. Advice About The Past / 對過去的建議
EN: This pattern often sounds like regret, criticism, or correction.

中：這個句型常帶有後悔、批評、或修正的語氣。

### D. Form Check / 形式檢查
EN: After **have**, use a past participle.

中：**have** 後面要接過去分詞。

- should have checked
- should have been corrected
- shouldn't have chosen

## Examples
- I should have checked the tense before answering.
- You should have used the passive form here.
- I shouldn't have rushed through the review test.
- This sentence should have been corrected earlier.

## Jason's Confusing Points
- EN: **Should have done** is past, not future.
- 中：**should have done** 是講過去，不是未來。
- EN: Passive version is **should have been done**.
- 中：被動版本是 **should have been done**。

## Wrong Answers
- Wrong: I should checked the answer.
- Correct: I should have checked the answer.
- Wrong: You shouldn't did that.
- Correct: You shouldn't have done that.
- Wrong: This should have corrected.
- Correct: This should have been corrected.

## One-line Summary
EN: **Should have done** means the better choice was in the past.

中：**should have done** = 過去本來應該做。
